# 🔌 คู่มือการต่อสาย ESP32 Table System

## 📋 สิ่งที่ต้องเตรียม

### Hardware ที่ต้องใช้:
- **ESP32 Development Board** (1 ตัว)
- **รีเลย์ 4CH 5V** (3 ตัว) 
- **เพจเจอร์/กระดิ่งเรียกโต๊ะ** (1 ตัว)
- **สายไฟเชื่อมต่อ** (สาย jumper)
- **แหล่งจ่ายไฟ 5V** (สำหรับ ESP32 และรีเลย์)

## 🔌 การต่อสาย ESP32 กับ รีเลย์ 
### 1. การต่อสายหลัก (ESP32 → รีเลย์)

```
ESP32 GPIO    →    รีเลย์ 16CH
─────────────────────────────────
GPIO 33       →    IN1  (เลข 0)
GPIO 14       →    IN2  (เลข 1) 
GPIO 27       →    IN3  (เลข 2)
GPIO 26       →    IN4  (เลข 3)
GPIO 25       →    IN5  (เลข 4)
GPIO 23       →    IN6  (เลข 5)
GPIO 22       →    IN7  (เลข 6)
GPIO 21       →    IN8  (เลข 7)
GPIO 19       →    IN9  (เลข 8)
GPIO 18       →    IN10 (เลข 9)
GPIO 13       →    IN11 (ปุ่ม CALL)
GPIO 32       →    IN12 (ปุ่ม Backspace)
```

### 2. การต่อสายเพจเจอร์/กระดิ่ง

```
รีเลย์ 16CH    →    เพจเจอร์/กระดิ่ง
─────────────────────────────────
COM (Common)  →    ขั้วบวกของเพจเจอร์
NO (Normally Open) → ขั้วลบของเพจเจอร์
```

### 3. การต่อสายไฟเลี้ยง

```
แหล่งจ่ายไฟ 5V   →    ESP32 และ รีเลย์ 16CH
─────────────────────────────────────────
ขั้วบวก (+)     →    VIN (ESP32) + VCC (รีเลย์)
ขั้วลบ (-)      →    GND (ESP32) + GND (รีเลย์)
```

## 🔧 ขั้นตอนการต่อสาย

### ขั้นตอนที่ 1: เตรียมอุปกรณ์
1. วาง ESP32 และรีเลย์ 16CH บน breadboard หรือ PCB
2. เตรียมสายไฟ jumper สำหรับการเชื่อมต่อ
3. เตรียมแหล่งจ่ายไฟ 5V (สำหรับทั้ง ESP32 และรีเลย์)
4. เตรียม Multimeter สำหรับวัดแรงดันไฟ

### ขั้นตอนที่ 2: ต่อสายไฟเลี้ยง
1. ต่อ GND ของ ESP32 กับ GND ของรีเลย์
2. ต่อแหล่งจ่ายไฟ 5V กับ VIN ของ ESP32 และ VCC ของรีเลย์
3. ต่อ GND ของแหล่งจ่ายไฟ กับ GND ของ ESP32 และรีเลย์
4. **วัดแรงดันไฟ:** ใช้ Multimeter วัดที่ VIN ของ ESP32 และ VCC ของรีเลย์
5. **ตรวจสอบแรงดัน:** ควรได้ 4.8V - 5.2V

### ขั้นตอนที่ 3: ต่อสายสัญญาณ
1. ต่อ GPIO 33 กับ IN1 (สำหรับเลข 0)
2. ต่อ GPIO 14 กับ IN2 (สำหรับเลข 1)
3. ต่อ GPIO 27 กับ IN3 (สำหรับเลข 2)
4. ต่อ GPIO 26 กับ IN4 (สำหรับเลข 3)
5. ต่อ GPIO 25 กับ IN5 (สำหรับเลข 4)
6. ต่อ GPIO 23 กับ IN6 (สำหรับเลข 5)
7. ต่อ GPIO 22 กับ IN7 (สำหรับเลข 6)
8. ต่อ GPIO 21 กับ IN8 (สำหรับเลข 7)
9. ต่อ GPIO 19 กับ IN9 (สำหรับเลข 8)
10. ต่อ GPIO 18 กับ IN10 (สำหรับเลข 9)
11. ต่อ GPIO 13 กับ IN11 (สำหรับปุ่ม CALL)
12. ต่อ GPIO 32 กับ IN12 (สำหรับปุ่ม Backspace)

### ขั้นตอนที่ 4: ทดสอบรีเลย์ (ก่อนต่อเพจเจอร์)
1. เปิดเว็บเบราว์เซอร์ไปที่ `http://192.168.1.181`
2. คลิก "Test All Relays" เพื่อทดสอบรีเลย์ทั้งหมด
3. ดูว่า LED indicator ของรีเลย์กระพริบหรือไม่
4. หลังจากรีเลย์ทำงานปกติแล้ว ค่อยต่อเพจเจอร์

### ขั้นตอนที่ 5: ต่อเพจเจอร์/กระดิ่ง (หลังจากรีเลย์ทำงานแล้ว)
1. ต่อ COM ของรีเลย์กับขั้วบวกของเพจเจอร์
2. ต่อ NO ของรีเลย์กับขั้วลบของเพจเจอร์
3. ตรวจสอบขั้วของเพจเจอร์ให้ถูกต้อง

## ⚠️ ข้อควรระวัง

### 1. การต่อสายไฟ
- **ตรวจสอบขั้วไฟให้ถูกต้อง** (บวก/ลบ)
- **อย่าให้ขั้วไฟสลับกัน** อาจทำให้อุปกรณ์เสียหาย
- **ใช้สายไฟที่เหมาะสม** กับกระแสที่ใช้

### 2. การต่อรีเลย์
- **ตรวจสอบแรงดันของรีเลย์** (5V - ง่ายกว่า 12V)
- **ตรวจสอบขั้ว COM, NO, NC** ให้ถูกต้อง
- **อย่าต่อขั้วไฟผิด** อาจทำให้รีเลย์เสียหาย

### 3. การต่อเพจเจอร์
- **ตรวจสอบขั้วของเพจเจอร์** (บวก/ลบ)
- **ตรวจสอบแรงดันที่เพจเจอร์ต้องการ**
- **อย่าต่อขั้วผิด** อาจทำให้เพจเจอร์เสียหาย

## 🔍 การทดสอบการต่อสาย

### 0. ทดสอบแรงดันไฟเข้า (ก่อนต่ออุปกรณ์)
**⚠️ สำคัญ: ต้องวัดแรงดันไฟก่อนต่ออุปกรณ์ทุกครั้ง**

#### **วัดแหล่งจ่ายไฟ 5V:**
```
Multimeter → แหล่งจ่ายไฟ 5V
─────────────────────────────
ขั้วบวก (+) → ขั้วบวกของแหล่งจ่ายไฟ
ขั้วลบ (-) → ขั้วลบของแหล่งจ่ายไฟ
```
- **แรงดันที่ควรได้:** 4.8V - 5.2V
- **ถ้าแรงดันต่ำกว่า 4.5V:** แหล่งจ่ายไฟไม่เพียงพอ
- **ถ้าแรงดันสูงกว่า 5.5V:** อาจเสียหายอุปกรณ์

#### **วัด ESP32 (หลังต่อไฟเลี้ยง):**
```
Multimeter → ESP32
─────────────────────
ขั้วบวก (+) → VIN pin ของ ESP32
ขั้วลบ (-) → GND pin ของ ESP32
```
- **แรงดันที่ควรได้:** 4.8V - 5.2V
- **ถ้าแรงดันต่ำ:** ตรวจสอบการต่อสายไฟเลี้ยง

#### **วัดรีเลย์ 16CH (หลังต่อไฟเลี้ยง):**
```
Multimeter → รีเลย์ 16CH
─────────────────────────
ขั้วบวก (+) → VCC pin ของรีเลย์
ขั้วลบ (-) → GND pin ของรีเลย์
```
- **แรงดันที่ควรได้:** 4.8V - 5.2V
- **ถ้าแรงดันต่ำ:** ตรวจสอบการต่อสายไฟเลี้ยง

### 1. ทดสอบ ESP32 (ก่อนต่อรีเลย์)
```cpp
// ทดสอบใน Serial Monitor
0  // ควรเห็น "Digit 0 pressed"
1  // ควรเห็น "Digit 1 pressed"
C  // ควรเห็น "CALL pressed"
B  // ควรเห็น "BACKSPACE pressed"
```

### 2. ทดสอบรีเลย์ (ไม่ต้องต่อเพจเจอร์)
**วิธีที่ 1: ใช้ Multimeter**
- วัดแรงดันที่ขั้ว COM และ NO
- เมื่อ ESP32 ส่งสัญญาณ HIGH ควรมีแรงดัน 5V
- เมื่อ ESP32 ส่งสัญญาณ LOW ควรไม่มีแรงดัน

**วิธีที่ 2: ใช้ Web Interface**
- เปิดเว็บเบราว์เซอร์ไปที่ `http://192.168.1.181`
- คลิก "Test All Relays"
- ดูว่า LED indicator ของรีเลย์กระพริบหรือไม่

### 3. ทดสอบเพจเจอร์ (หลังจากรีเลย์ทำงานแล้ว)
- เปิดเว็บเบราว์เซอร์ไปที่ `http://192.168.1.181`
- คลิก "Test Table 5" เพื่อทดสอบการเรียกโต๊ะ
- เพจเจอร์ควรส่งเสียงเรียกโต๊ะ 5
- ตรวจสอบว่าเสียงชัดเจนและถูกต้อง

### 4. ทดสอบด้วย Multimeter (สำหรับรีเลย์ 16CH)
- วัดแรงดันที่ขั้ว COM และ NO ของรีเลย์
- เมื่อ ESP32 ส่งสัญญาณ HIGH ควรมีแรงดัน 5V
- เมื่อ ESP32 ส่งสัญญาณ LOW ควรไม่มีแรงดัน
- ตรวจสอบว่าแรงดันเปลี่ยนตามสัญญาณหรือไม่

## 🛠️ การแก้ไขปัญหา

### ปัญหา: รีเลย์ไม่ทำงาน
**สาเหตุ:** 
- สายไฟหลุด
- แรงดันไม่เพียงพอ
- ขั้วไฟผิด

**วิธีแก้:**
- ตรวจสอบการต่อสายไฟ
- ตรวจสอบแรงดัน 5V
- ตรวจสอบขั้วไฟ

### ปัญหา: เพจเจอร์ไม่ส่งเสียง
**สาเหตุ:**
- สายไฟหลุด
- ขั้วเพจเจอร์ผิด
- เพจเจอร์เสีย

**วิธีแก้:**
- ตรวจสอบการต่อสายเพจเจอร์
- ตรวจสอบขั้วเพจเจอร์
- ทดสอบเพจเจอร์ด้วยแหล่งจ่ายไฟโดยตรง

### ปัญหา: ESP32 ไม่ส่งสัญญาณ
**สาเหตุ:**
- โค้ดผิด
- GPIO pin ผิด
- การตั้งค่าผิด

**วิธีแก้:**
- ตรวจสอบโค้ด
- ตรวจสอบ GPIO pin
- ตรวจสอบการตั้งค่า

## 📞 การติดต่อ

หากมีปัญหาการต่อสายหรือการทำงาน สามารถสอบถามได้ที่:
- ตรวจสอบ Serial Monitor ของ ESP32
- ดู IP address ที่แสดงใน Serial Monitor
- ทดสอบผ่านเว็บเบราว์เซอร์ที่ `http://[IP_ADDRESS]`
